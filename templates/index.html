¿Qué es RASPIARM?

RASPIARM surge de la idea de aprovechar la potencia de las tarjetas Raspberry Pi en combinación con otras tecnologías y dispositivos, tales como Arduino, Python, HTML, CSS y JavaScript, para la automatización de procesos. Este proyecto se enfoca en la implementación de un brazo mecánico y un elevador automatizado, aprovechando la versatilidad y capacidad de programación de estas herramientas para crear soluciones eficientes y personalizadas.

Hola somos; Axel, Adam, Oskar, Melissa, Fernando y Saharí

Somos estudiantes de octavo cuatrimestre de Ingeniería en Sistemas Computacionales y este proyecto representa una oportunidad invaluable para fortalecer nuestras habilidades técnicas. Estamos comprometidos con la aplicación práctica de las tecnologías emergentes para ofrecer soluciones innovadoras. Este proyecto no solo nos permite consolidar nuestros conocimientos, sino también nos brinda la experiencia de trabajar en un entorno real, preparándonos para enfrentar los desafíos del mundo laboral con confianza y creatividad.
